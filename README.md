## Pre-requisites

### Neovim - [brew](https://brew.sh)

```bash
$ brew install neovim
```

### NvChad Configuration

```bash
$ rm -rf ~/.cache/nvim ~/.local/share/nvim/ ~/.config/nvim
$ git clone https://github.com/elulcao/NvChad-custom.git --depth 1 $HOME/.config/nvim
$ nvim
```
